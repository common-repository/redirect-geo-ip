Redirect via Geo IP plugin automatically determines visitors locations and redirects them to the correct store.

== Description ==
GeoIP redirect plugin helps change the store based on the user's IP Address. This plugin can be used where there are stores. In these cases, based on customer IP, the most convenient store is selected. The desired URL can be specified to manage the redirections.
The plugin automatically determines visitors location and redirects them to the correct store. With the help of this plugin, the admin users can improve the customer's shopping experience by showing localized information.

== Installation ==
1. Upload the plugin files to the /wp-content/plugins/ directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the /Plugins/ screen in WordPress
3. Settings/Woocommerce settings -> Redirect via Geo IP
3.1. Option for visitor country choose
3.2. Redirect to URL to store

== Features of Plugin ==
Redirect users to a specific store view.
